# Sales Data Analysis & Dashboard (Python + Google Sheets)

This project demonstrates an end-to-end data analysis workflow, combining Python and Google Sheets to transform raw sales data into actionable business insights and interactive visual dashboards.

1. Project Overview
The goal of this project is to analyze sales data, identify key trends, and present insights in a clear and visually engaging way. It covers the full data pipeline from data cleaning and analysis to reporting and dashboard creation.

2. Tools & Technologies**
- Python (pandas, numpy, matplotlib/seaborn)
- Google Sheets (pivot tables, charts, formulas)
- upyter Notebook
- 
3. Key Features
- Data cleaning and preprocessing using Python
- Exploratory data analysis (EDA)
- Sales trend analysis (monthly, regional, product-level)
- Visualization of key metrics
- Interactive dashboard built in Google Sheets
  
4. Insights Generated
- Identification of top-performing products
- Revenue trends over time
- Regional sales performance comparison
- Key business patterns and opportunities

5. Dashboard
The final dashboard is created in Google Sheets, providing an interactive way to explore the data using charts, filters, and summary metrics.

6. Project Structure
- data/ – raw and cleaned datasets
- notebooks/ – analysis notebooks
- scripts/ – data processing scripts
- dashboard/ – Google Sheets link
- visuals/ – generated charts
  
7. Conclusion
This project highlights practical data analysis skills and demonstrates how to communicate insights effectively using both Python and spreadsheet tools.
